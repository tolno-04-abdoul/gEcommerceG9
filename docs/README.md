## 🎯 Présentation du projet

Ce projet consiste à mettre en place une base de données Oracle pour la **gestion d'un site e-commerce**. Il inclut :

- La gestion des clients, produits, commandes et paiements
- L'optimisation des performances (index, vues matérialisées)
- L'automatisation des processus (triggers, batchs)
- La sauvegarde et la restauration (RMAN, Data Pump)
